# BCM94360CD Linux v1.0.0

Initial stable release of the offline BCM94360CD / BCM4360 Wi-Fi installer for
Ubuntu 26.04 LTS.

## Tested

- Mac Pro Late 2013 (`MacPro6,1`, A1481)
- Broadcom BCM94360CD / BCM4360
- PCI ID `14e4:43a0`
- Ubuntu 26.04 LTS amd64
- Kernel `7.0.0-14-generic`

## Included

- Ubuntu `broadcom-sta-dkms` `6.30.223.271-29ubuntu1`
- Matching `7.0.0-14-generic` kernel headers
- Complete offline DKMS build dependency set
- SHA-256 integrity verification
- Hardware, OS, architecture, and kernel validation
- Automatic `wl` module build and interface verification

## Release Assets

- `bcm94360cd-linux-ubuntu-26.04-7.0.0-14-generic.tar.gz`
- `bcm94360cd-linux-ubuntu-26.04-7.0.0-14-generic.tar.gz.sha256`

The archive is kernel-specific. Build a new archive with `build-bundle.sh` if
the target system runs a different kernel.
